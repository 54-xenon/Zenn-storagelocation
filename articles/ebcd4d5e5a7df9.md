# 擬似要素とは
擬似要素とは指定した特定の要素にスタイルをつけることができるものです。よく、CSSで::afterや::beforeを見かけることがあルトもいます。

## HTMl
擬似要素を使うためにあHTMLの一部に何かを加える必要はありません。いつも通り、HTNLを書いていくだけです。(HTMLに関しての説明は今回省きます。)
```html
<!DOCTYPE html>
<html>
<head>
</head>
<body>
    <div>
        <h2>タイトル</h2>
    </div>
</body>
</html>
```
